myJAM doesn't run on Microsoft Internet Explorer

due to lacking or erroneous support of CSS

Please install one of the following browsers:

  • Firefox 2.x or higher (available for Linux, Mac and Windows); Download: http://www.mozilla.org)
  • Safari 3.x or higher (available for Mac and Windows); Download: http://www.apple.com/de/safari/
  • Opera 9.x or higher (available for Linux, Mac and Windows); Download: http://www.opera.com/
  • Konqueror 3.5.x or higher (available for Linux); Download: http://www.konqueror.org/
Furthermore, the Internet Explorer has problems to handle XML Strict embedded Flash animations. See http://alistapart.com/articles/flashsatay for more information.
