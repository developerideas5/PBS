WampServer, Web development platform in Windows

WampServer is a Web development platform for Windows dynamic Web applications using the Apache 2 server, PHP scripting language and MySQL database. It also possesses PHPMyAdmin to easily manage your databases.

experimenting WampServer
3.2K

Start with WampServer

As you will see, WampServer installs easily and its very intuitive to configure it very quickly (without touching the configuration files).

Installation

  • Double-click the downloaded file and follow the instructions. Everything is managed by the installer of WampServer. By default WampServer comes with the latest versions of Apache, MySQL and PHP.
  • Once installed, you can manually add additional versions of Apache, PHP or MySQL (only compiled VC9, VC10 or VC11). The explanations for the data will be on the  forum .
  • Each version of Apache, MySQL and PHP has its own configuration and its own files (data for MySQL).
Use WampServer:
  • automatic creation of a "www" directory during installation (typically c: \ wamp \ www).
  • Create a subdirectory for your project and drop there your PHP files
  • Click the link "Localhost" the WampServer menu or open your favorite browser and go to http: // localhost

features

WampServer also has a "TrayIcon" allowing you to manage and easily configure your servers, without touching the configuration files.

Left click on the WampServer icon, you can:

  • Manage Apache and MySQL services
  • switch to online / offline (accessible to all or restricted to localhost)
  • Install and change the version of Apache, MySQL and PHP
  • Manage your server configuration settings
  • Access to your logs
  • Access configuration files
  • Create alias..et more

Right click :

  • Change the language of the WampServer menu and go directly to that page

Downloads

WampServer is free (under GPL license) in two versions: 32 and 64 bit. Wampserver 2.5 is not compatible with Windows XP, even SP3 or Windows Server 2003. Older versions of Wampserver are at  Sourceforge .
